== Interactive Endpoint Demo

Created for SpreeConf DC, 2013 for my talk on the Spree Integrator.


== Steps to follow along:


1. Clone this repo, and move into directory:

    git clone git://github.com/BDQ/demo_endpoint.git; cd demo_endpoint;

2. Install dependencies:

    bundle install

3. Start server:

    bundle exec rackup -s thin -p 9000

4. Open your favourite browser, and go to:

    http://localhost:9000/

5. Listen to me carefully!
